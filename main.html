Create Your Portfolio Page

A lot of developers use their portfolio websites to display their skills and resume. In this task, you will create a simple portfolio page with the following requirements:

Requirement Description
Image Insert your image using the img tag.
Name Write your name using a heading element (e.g., h1 tag).
LinkedIn Profile Provide a link to your LinkedIn profile using the a (anchor) tag. The label of the link should contain at least one character.
Skills Section Create a section with the id skills to list your skills. This section must contain text.
Projects Section Create a section with the id projects and list your projects in an unordered list. Each project must be listed in a li tag and contain text.

Note: Styling with CSS is optional for this task, but you may choose to add styling to enhance the appearance of the page.
